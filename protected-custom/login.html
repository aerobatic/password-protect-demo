Password Protect Demo

This is a custom password-protect login page. You can make this page look like anything you like.

Example

Access to this site is restricted to our clients. Since this is just a demo, the password is aerobatic.

You can intentionally try an incorrect password to see error handling in action. After 5 failed logins within a 5 minute period, a 403 Forbidden error page will be returned which you can also customize with the custom-errors plugin.

To implement basic auth in your own website, see the Aerobatic documentation.
